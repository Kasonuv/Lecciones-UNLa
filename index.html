Curvas Características de Operación: de la probabilidad al plan de muestreo

Esta herramienta guía al estudiante en la construcción, lectura e interpretación de una CCO. El objetivo no es memorizar una curva, sino comprender cómo se calcula la probabilidad de aceptar un lote cuando la decisión se toma a partir de una muestra.

Idea central: en muestreo por atributos la variable aleatoria es el número de unidades defectuosas encontradas en la muestra. Por eso siempre trabajamos con una variable discreta.

Primera idea clave

La CCO parte de una variable discreta

En un plan de muestreo por atributos, cada unidad inspeccionada se clasifica como defectuosa o no defectuosa. La variable que se observa es:

X = cantidad de unidades defectuosas encontradas en la muestra

Como se trata de una cantidad contable de unidades, X solo puede tomar valores enteros:

X = 0, 1, 2, 3, ..., n
No tiene sentido decir que en la muestra aparecieron 2,7 unidades defectuosas. La proporción defectuosa del lote puede expresarse como decimal o porcentaje, pero el número de defectuosas observadas en la muestra es discreto.

Regla de decisión

Un plan simple se define por dos números:

SímboloSignificado
nTamaño de la muestra
cNúmero máximo de defectuosas permitido para aceptar el lote

El lote se acepta cuando:

X ≤ c
Segunda idea clave

¿Para qué se usan los modelos de probabilidad?

Los modelos de probabilidad se usan para calcular esta pregunta:

Si el lote tuviera una determinada proporción real de defectuosos, ¿cuál sería la probabilidad de que la muestra cumpla el criterio de aceptación?

Es decir:

Pa = P(X ≤ c)

Donde Pa es la probabilidad de aceptación del lote. Como X es discreta, el cálculo se hace sumando probabilidades puntuales:

P(X ≤ c) = P(X=0) + P(X=1) + ... + P(X=c)

Modelo binomial

Se usa como aproximación práctica cuando el lote es grande respecto de la muestra, o cuando puede considerarse que la probabilidad de defecto permanece aproximadamente constante.

X ~ Binomial(n, p)
Pa = Σx=0c C(n,x) px(1-p)n-x

p representa la proporción defectuosa real del lote.

Modelo hipergeométrico

Se usa cuando el lote es finito y la muestra se extrae sin reposición. Es más exacto cuando la muestra no es despreciable frente al lote.

X ~ Hipergeométrica(N, D, n)
Pa = Σx=0c [C(D,x) C(N-D,n-x)] / C(N,n)

N es el tamaño del lote y D la cantidad real de defectuosas en el lote.

Procedimiento

Paso a paso para construir una CCO

Definir el plan de muestreo

Fijar n y c. Ejemplo: n = 50, c = 2. El lote se acepta si X ≤ 2.

Elegir niveles posibles de calidad del lote

Tomar varios valores de p: 0%, 1%, 2%, 3%, 5%, 8%, 10%, etc. Cada valor es una hipótesis sobre la proporción real de defectuosos del lote.

Calcular la probabilidad de aceptación

Para cada valor de p, calcular Pa = P(X ≤ c). Como X es discreta, se suman las probabilidades desde X = 0 hasta X = c.

Armar la tabla de puntos

Cada par (p, Pa) será un punto de la CCO.

Graficar la curva

En el eje horizontal se coloca la proporción defectuosa del lote; en el eje vertical, la probabilidad de aceptación.

Resultado esperado: una curva descendente. A medida que aumenta la proporción defectuosa del lote, debería disminuir la probabilidad de aceptarlo.

Interpretación

Cómo se lee una CCO

Lectura correcta: si para p = 3% la curva muestra Pa = 0,81, eso significa que, si muchos lotes tuvieran realmente 3% de defectuosos, este plan aceptaría aproximadamente el 81% de esos lotes.
Lectura incorrecta: no significa que el lote tenga 81% de unidades buenas. La CCO no informa directamente la composición de una muestra particular; informa la probabilidad de aceptación bajo un plan.
